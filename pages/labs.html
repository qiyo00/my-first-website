Lab 2

15/January/2026

What I Learned

CSS Layout Techniques

In this lab, I explored two powerful CSS layout systems:

  • Flexbox - Perfect for one-dimensional layouts like navigation bars
  • CSS Grid - Ideal for two-dimensional layouts like image galleries

Responsive Design

I learned how to make websites adapt to different screen sizes using media queries. This ensures my site looks great on phones, tablets, and desktops.

Screenshots

Reflection

Besides Flex box and CSS grid, I learned the difference between px and rem units: rem (Root em) is relative to the browser's default font size (usually 16px). Using rem instead of fixed pixels makes the website more accessible and responsive, as it scales automatically if a user adjusts their browser settings. I also learned how to use CSS variables (:root) to manage colors efficiently, making it much easier to "theme" my website later.

The hardest concept to visualize was the Parent-Child relationship in CSS layout, specifically regarding "Full Bleed" (full-width) designs. At first, I struggled to understand why my Hero image had white borders around it even though I set width: 100%. It was hard to visualize that because my main tag (the parent) had padding and max-width limitations, the hero section (the child) was trapped inside that "invisible cage." Understanding that I needed to physically move the code structure (taking the Hero section out of the main container) was challenging.

The "Squished" Navbar Challenge: I faced a frustrating issue where my navigation bar collapsed into a tiny block in the center of the screen. I was surprised to learn that this was caused by setting display: grid and place-items: center on the . It taught me that generic centering commands on the body affect everything, including elements like headers that are supposed to span the full width. I also learned the "Container Strategy": allowing the main or body to be full-width (for things like the Hero image), but creating a separate internal div (like .content-container) to hold the text and gallery. This hybrid approach allows me to have both immersive full-screen images and readable, centered text on the same page.
